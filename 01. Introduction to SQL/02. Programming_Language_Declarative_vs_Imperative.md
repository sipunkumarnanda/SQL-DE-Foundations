# 🤖 What is a Programming Language?

## 📖 Definition

A **Programming Language** is a language used by humans to communicate instructions to a computer.

Simply put,

> A programming language tells the computer **what to do**.

Think of it as a translator between humans and computers.

---

# 🧠 Why Do We Need Programming Languages?

Imagine you want your computer to display:

```
Hello World
```

Can you simply tell the computer:

> "Please print Hello World."

❌ No.

Because computers **don't understand English.**

A computer understands only:

```
0 and 1
```

This is called **Machine Language (Binary Language).**

Example:

```
01001011 10100101 11101010
```

Can humans easily read this?

❌ Absolutely not.

So we created programming languages.

Programming languages allow humans to write instructions in a readable format, which are then translated into machine code.

# 🏗️ Two Ways to Give Instructions

There are two major styles of programming.

## 1️⃣ Imperative Programming

## 2️⃣ Declarative Programming

Understanding these two styles is one of the most important concepts before learning SQL.

---

# 🔵 Imperative Programming

## 📖 Definition

Imperative Programming tells the computer:

> **HOW to do something.**

You give the computer **step-by-step instructions.**

You control every step.

---

## 🍵 Example 1: Making Tea

Imagine your mother asks you to make tea.

She says:

1️⃣ Take a pan.

2️⃣ Pour water.

3️⃣ Add tea leaves.

4️⃣ Add sugar.

5️⃣ Boil for 5 minutes.

6️⃣ Add milk.

7️⃣ Serve.

Notice something?

She explained every single step.

This is Imperative Programming.

---

## 💻 Java Example

```java
int sum = 0;

for(int i = 1; i <= 5; i++) {
    sum += i;
}

System.out.println(sum);
```

Here, we tell Java:

* Create a variable
* Start loop
* Increase i
* Add numbers
* Print result

Everything is explained step by step.

This is Imperative Programming.

---

## 📦 Real-Life Example

Imagine asking someone to clean your room.

Instead of saying

> "Clean the room."

You say

* Pick up the clothes.
* Fold them.
* Put them in the cupboard.
* Sweep the floor.
* Mop the floor.
* Arrange the books.

You described **HOW** to do the work.

---

# ⭐ Characteristics of Imperative Programming

✅ Focuses on **HOW**

✅ Step-by-step instructions

✅ Programmer controls execution

✅ More detailed

✅ Gives full control

Examples:

* Java
* C
* C++
* JavaScript
* Python (mostly imperative)

# 🟢 Declarative Programming

## 📖 Definition

Declarative Programming tells the computer:

> **WHAT you want, not HOW to do it.**

You describe the goal.

The system figures out the implementation.

---

## 🍔 Restaurant Example

Imagine you're in a restaurant.

Do you tell the chef:

* Chop onions
* Heat oil
* Add salt
* Fry vegetables
* Add spices

❌ No.

You simply say

> "One Paneer Butter Masala."

The chef decides **how** to prepare it.

This is Declarative Programming.

---

## 🌍 SQL Example

Suppose we have a Student table.

| ID | Name  | Fees  |
| -- | ----- | ----- |
| 1  | Rahul | 25000 |
| 2  | Ankit | 22000 |
| 3  | Priya | 30000 |

Now we write:

```sql
SELECT * FROM Students;
```

Did we tell SQL:

* Open database
* Read disk
* Find table
* Scan rows
* Allocate memory
* Return data

❌ No.

We simply said:

> "Give me all students."

The database engine decides **how** to retrieve the data efficiently.

This is Declarative Programming.

---

# 🔥 Another SQL Example

Suppose we write:

```sql
SELECT name
FROM Students
WHERE fees > 25000;
```

What did we tell SQL?

We simply said:

👉 Give me students whose fees are greater than ₹25,000.

Did we explain:

* Which index to use?
* Which page to read?
* Which algorithm to choose?
* Which memory block to access?

❌ No.

The database optimizer decides the best execution plan.

That is why SQL is called a Declarative Language.

---

# 🤔 Why is SQL Declarative?

Because SQL focuses on the result.

Example:

```sql
SELECT * FROM Employees;
```

We only specify:

> "I want all employees."

The database internally decides:

* Which storage pages to read
* Whether to use an index
* Which join algorithm to use (if needed)
* How to optimize the query
* How to use memory and CPU efficiently

The developer doesn't need to manage these low-level details.

---

# ⚖️ Imperative vs Declarative Programming

| 🟦 Imperative Programming         | 🟩 Declarative Programming              |
| --------------------------------- | --------------------------------------- |
| Focuses on **HOW**                | Focuses on **WHAT**                     |
| Programmer writes every step      | Programmer describes the desired result |
| More control                      | Less manual control                     |
| Usually longer code               | Usually shorter code                    |
| Developer manages execution       | System manages execution                |
| Example: Java, C, C++, JavaScript | Example: SQL, HTML, CSS                 |

# 🧠 Interview Question

## ❓ Is SQL a Programming Language?

### ✅ Best Answer

Yes, SQL (Structured Query Language) is considered a **domain-specific declarative language** used for managing and querying relational databases. Unlike general-purpose imperative languages such as Java or C++, SQL focuses on describing **what data is required**, while the database management system determines **how** to retrieve or modify that data efficiently.

---

# 🎤 FAANG Interview Answer

### ❓ What is the difference between Imperative and Declarative Programming?

> **Imperative programming** focuses on **how** to solve a problem by providing step-by-step instructions to the computer. Languages like Java, C, and C++ follow this approach. **Declarative programming**, on the other hand, focuses on **what** result is required without specifying the execution steps. SQL is a declarative language because developers specify the desired data, and the database engine decides the most efficient execution plan. This abstraction improves productivity while allowing the database optimizer to maximize performance.

---

# 🎯 Key Takeaways

✅ A programming language is a medium through which humans communicate instructions to computers.

✅ Computers understand only machine language (binary), so programming languages act as translators.

✅ Imperative programming tells the computer **HOW** to perform a task through explicit step-by-step instructions.

✅ Declarative programming tells the computer **WHAT** result is desired, leaving the implementation details to the system.

✅ SQL is a **declarative language** because developers specify the required data, while the database engine determines the most efficient way to execute the query.

✅ Understanding the difference between imperative and declarative programming is essential for mastering SQL and explaining its design in technical interviews.
